Предназначена для вторичного подтверждения операций в сервис облачной подписи для варианта получения кода по SMS или Email.

Порядок работы

  • В поле сертификат указывается информация о сертификате, используемый при операции.
  • В поле способ подтверждения можно изменить текущий способ на любой доступный в списке выбора. 
  • В поле документы указывается список документов, с которыми проводится операция. Доступно только для операции создание электронной подписи.
  • В поле код подтверждения необходимо ввести одноразовый пароль, полученный в SMS или Email. 
  • Кнопка повторить позволяет запросить получение повторного временного пароля. Внимание получение повторного одноразового пароля ограничено по времени.
  • В нижней части выводиться время, оставшееся для окончания процедуры подтверждения операции
