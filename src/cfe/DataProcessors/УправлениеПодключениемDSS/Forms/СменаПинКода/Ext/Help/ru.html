Предназначена для авторизации пользователя на сервисе облачной подписи.

Форма предназначена для всех способов первичной авторизации указанной в карточке учетной записи, кроме Авторизация через Web-форму. Вызывается в момент выполнения первой операции с помощью сервиса облачной подписи, последующие операции такой авторизации не требуют, кроме способа Логин и пароль с дополнительным подтверждением. В этом случае в зависимости от настроек сервиса может потребоваться повторная авторизация по истечению срока действия маркера безопасности.

Настройка подключения

  • В поле логин указывается имя учетной записи облачной подписи.
  • В поле сервер указывается, какой сервер используется для подключения. 
  • В поле сертификат указывает статус загруженного сертификата авторизация выданного пользователю в момент регистрации провайдером сервиса. Это поле доступно только для способа Сертификат авторизации
  • В поле пароль необходимо ввести пароль учетной записи. 
  • Флажок сохранить позволяет сохранить пароль БД после успешной авторизация и незапрашивать его у пользователя при последующих подключениях к сервису.
