Предназначена для вторичного подтверждения операций в сервис облачной подписи с помощью мобильного приложения.

Для использования этого способа подтверждения на мобильное устройство на платформах iOS или Android должно быть установлено мобильное приложение myDSS или AirKey DSS. Дополнительно мобильное устройство с установленным приложением должно быть зарегистрировано в сервисе облачной подписи.

Порядок работы

  • В поле сертификат указывается информация о сертификате, используемый при операции.
  • В поле способ подтверждения можно изменить текущий способ на любой доступный в списке выбора. 
  • В поле документы указывается список документов, с которыми проводится операция. Доступно только для операции создание электронной подписи.
  • Процесс подтверждения сопровождается анимацией, программа периодически опрашивает статус подтверждения в сервисе и в случае успеха автоматически закрывается форма.
  • В нижней части выводиться время, оставшееся для окончания процедуры подтверждения операции
